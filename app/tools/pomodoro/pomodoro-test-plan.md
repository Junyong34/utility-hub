## Affected Pages/Routes

- `/tools/pomodoro` — 첫 진입 Quick Start, mode toggle, canvas stage, control bar
- `/tools` — 뽀모도로 카드 노출과 tool navigation

## Key Interactions to Verify

- 첫 방문 시 25분 기본 타이머가 즉시 보이는지
- 입력 없이 `Start`가 가능한지
- `Pause` → `Resume` → `Reset` 흐름이 안정적인지
- Quick Start와 Task Mode 전환 시 세션이 의도대로 유지되는지
- Task 생성/선택/완료/삭제 흐름이 정상인지
- 완료 시 animation, record save, task complete가 한 번만 발생하는지

## Edge Cases

- hidden tab에서 오래 머문 뒤 복귀
- 새로고침 후 active session 복구
- localStorage JSON 손상
- localStorage write 실패
- sound autoplay 차단
- fullscreen 거부 또는 미지원 브라우저
- active task 삭제 중 세션 진행
- 키보드만으로 조작
- 모바일 viewport에서 panel open/close

## Critical Paths

- Quick Start 첫 방문 → Start → 완료 → 기록 저장 → 다시 시작
- Task Mode 빈 상태 → 첫 Task 생성 → Start → 완료 → task completed
- 진행 중 새로고침 → 세션 복구 → 완료
- 권한 거부(sound/fullscreen) 상태에서도 세션 완료 경험 유지
