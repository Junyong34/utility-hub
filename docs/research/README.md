# 리서치 문서 인덱스

기준일: `2026-04-07`

이 디렉터리는 육아형 리브랜딩에 필요한 사전 조사 문서를 모아 둔 곳입니다. 현재는 `장소 리서치`와 `공공데이터 리서치`를 분리해 관리합니다.

## 디렉터리 구성

- [`places/README.md`](./places/README.md)
  서울·경기·인천 아동 동반 장소, 공공형 시설, 가족 동반 카페, 사설 키즈카페, 영아 친화 베이비카페 리서치
- [`public-data/README.md`](./public-data/README.md)
  육아·어린이 관련 공공데이터 목록과 서비스 아이디어 리서치

## 공통 원칙

- 운영 사실은 `공식 링크` 기준으로 확인합니다.
- `네이버지도`는 위치 확인용 편의 링크로만 사용합니다.
- 공식 URL을 확정하지 못한 항목은 `추가 검증 필요` 또는 `보류 후보`로 분리합니다.
- 문서 기준일과 행 단위 `검증일`은 분리해서 읽습니다.

## 읽는 순서

1. 장소 허브 설계가 목적이면 [`places/README.md`](./places/README.md)부터 읽습니다.
2. 데이터 기반 서비스 기획이 목적이면 [`public-data/parenting-public-data-opportunities.md`](./public-data/parenting-public-data-opportunities.md)를 먼저 읽습니다.
3. 실제 구현 전에는 공식 링크와 최신 운영 여부를 한 번 더 재검증합니다.

간단한 결론: `docs/research`는 `/places`, `/benefits`, 후속 도구 기획 전에 확인하는 1차 조사 저장소입니다.
