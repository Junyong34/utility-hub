# 육아/어린이 공공데이터 활용 리서치

기준일: `2026-04-07`

## 문서 목적

부모가 실제로 쓸 수 있는 서비스 기준으로, 육아/어린이 관련 공공데이터가 무엇인지와 그 데이터로 어떤 기능을 만들 수 있는지 정리한다. 민간 재가공 정보는 참고만 하고, 본문은 공식 URL 위주로만 구성한다.

## 검증 원칙

- 공식 기관/공공 포털의 원문 페이지를 우선한다.
- 지역별로 흩어진 데이터는 하나의 대표 공식 URL로 묶고, 필요하면 지자체 예시를 함께 적는다.
- 운영 여부, 갱신성, 필드 구성을 확인할 수 없는 항목은 `추가 검증 필요`로 보낸다.
- 예시로 나온 GitHub 저장소는 서비스 발상 참고용일 뿐, 출처로 삼지 않는다.

## 신뢰도 표기 규칙

- `official_verified`: 공식 페이지에서 데이터 성격과 활용 가능성을 직접 확인함
- `semi_verified`: 공식 경로는 맞지만 접근이 메뉴형이거나 지역별 예시 확인이 필요한 경우
- `discovery_only`: 수요는 있지만 이번 차수에서 공식 URL을 확정하지 못한 경우

## 빠르게 볼 결론

- 가장 바로 쓸 수 있는 축은 `응급의료`, `예방접종`, `어린이집 현황`, `보육 인증`, `성장도표`, `어린이보호구역/놀이시설`, `출생 통계`다.
- 부모 실사용 기준으로는 `병원 찾기`, `예방접종 가능 기관 찾기`, `어린이집 비교`, `성장 체크`, `안전 지도`, `출생 추이 대시보드`가 우선순위가 높다.
- `아이 이름 순위`는 수요는 있지만 이번 차수에서 공식 데이터를 확정하지 못해 별도 보류가 맞다.

## 서비스 아이디어

### 의료와 응급

- `달빛어린이병원/야간 소아과 찾기`: 지역, 오늘 운영 여부, 야간 진료 가능 시간, 응급실 여부를 묶어서 보여준다.
- `응급실 혼잡도 지도`: 응급의료기관의 실시간 가용병상과 위치를 함께 보여준다.

### 예방접종

- `어린이 예방접종 가능한 병원 찾기`: 백신 종류, 요일제 운영, 지역, 거리 필터를 붙여 부모가 바로 예약 후보를 좁히게 한다.
- `예방접종 일정 확인 도우미`: 연령대별 필수 접종을 기준으로 놓친 접종을 체크하게 한다.

### 성장과 건강검진

- `공식 성장도표 기반 성장 체크`: 월령과 키/몸무게를 넣으면 백분위와 해석을 보여준다.
- `영유아 검진기관 찾기`: 건강검진 대상 시기와 가까운 검진기관을 연결한다.

### 보육과 돌봄

- `어린이집 수급 대시보드`: 지역별 어린이집 수, 정원, 현원, 공공형 여부, 평가인증 여부를 한 화면에 보여준다.
- `정원 대비 현원 알림`: 현원 비율이 높은 어린이집이나 여유가 큰 어린이집을 비교한다.

### 안전과 장소

- `스쿨존 안전 지도`: 어린이보호구역 사고 통계와 주변 놀이시설을 함께 본다.
- `실내/실외 놀이시설 탐색`: 놀이시설 현황을 실내외, 민공구분, 설치장소로 나눈다.

### 출생과 지역 수요

- `지역별 육아 수요 리포트`: 출생아 수와 합계출산율 추이를 지역별로 묶어 콘텐츠와 허브 우선순위를 정한다.
- `부모용 간단 대시보드`: 출생 추이, 보육 인프라, 의료 접근성을 한 번에 요약한다.

## 구현 우선순위 제안

1. `응급의료 + 달빛어린이병원 + 예방접종`부터 붙인다. 가장 즉시성이 높다.
2. `어린이집 현황 + 정현원 + 공공형/평가인증`을 합쳐 비교 기능을 만든다.
3. `성장도표 + 영유아 검진기관`을 붙여 건강 체크 흐름을 만든다.
4. `어린이보호구역 + 놀이시설`로 안전 지도를 확장한다.
5. 마지막으로 `출생 통계`를 묶어 지역별 콘텐츠 우선순위에 사용한다.

## 추가 검증 필요 항목

- `아이 이름 순위`: 이번 차수에서 공식 오픈데이터/API 소스를 확정하지 못했다.
- `전국 단일 키즈카페 공공데이터`: 수요는 있지만 공식적으로 전국을 한 번에 커버하는 데이터는 확인이 필요하다.
- `NHIS 영유아 검진기관`은 공식 포털은 확인했지만, 서비스화할 때는 정확한 메뉴 경로 또는 별도 조회 경로를 더 고정하는 편이 안전하다.

간단한 결론: 지금 바로 서비스화하기 좋은 데이터는 `응급의료`, `예방접종`, `어린이집`, `성장도표`, `안전`, `출생 통계`다. 반면 `아이 이름 순위`처럼 흥미는 있지만 공식 소스가 덜 고정된 항목은 이번 차수에서 빼는 편이 맞다.
