# 육아 공공데이터 리서치

기준일: `2026-04-07`

이 디렉터리는 육아/어린이 관련 공공데이터를 `공식 URL 기준`으로 정리한 리서치 문서 모음입니다. 장소 리서치처럼 후보를 많이 모으는 목적이 아니라, `어떤 공공데이터가 실제 서비스로 이어질 수 있는지`를 먼저 확인하는 용도입니다.

## 파일 구성

- [`parenting-public-data-opportunities.md`](./parenting-public-data-opportunities.md)

## 문서 정책

- `official_verified`: 공식 기관/공공 포털에서 직접 확인한 데이터
- `semi_verified`: 공식 기관 경로는 맞지만, 구체 접근 경로가 메뉴형이거나 보조 확인이 더 필요한 데이터
- `discovery_only`: 아이디어는 유효하지만 이번 차수에서 공식 URL을 확정하지 못한 항목
- `exclude`: 공식 근거가 약하거나 발행용 데이터셋으로 쓰기 어려운 항목

이 디렉터리의 메인 문서는 `official_verified`와 `semi_verified`만 본문에 두고, `discovery_only`는 `추가 검증 필요` 섹션으로 분리합니다.

## 읽는 법

- 공공데이터가 있으면 바로 서비스가 되는 것은 아닙니다.
- 이 문서는 `데이터 존재 여부`, `갱신 가능성`, `부모가 실제로 쓸 수 있는지`를 같이 봅니다.
- 예시로 언급되는 GitHub 저장소는 참고 맥락일 뿐, 본문 판단 근거는 아닙니다.

간단한 결론: 이 디렉터리는 `육아/어린이 서비스 아이디어를 공식 데이터로 연결하는 출발점`입니다. 신뢰도는 공식 포털과 공식 기관 페이지에만 둡니다.
