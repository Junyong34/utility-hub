# 서울·경기·인천 문화센터·어린이 체험시설 조사

- 기준일: `2026-07-13`
- 대상: 수도권의 트니트니 운영 문화센터·직영센터와 상설 어린이 체험시설
- 집계 단위: 같은 시설의 강좌가 여러 개여도 시설 1곳을 1건으로 계산
- 제외: 단기 행사, 일반 상업 키즈카페, 단순 놀이터·도서관, 운영 신호를 공식 출처에서 확인하지 못한 후보

## 먼저 보는 결론

`어린이 체험시설`은 법정 단일 업종이 아니고 네트워크끼리 같은 건물이나 시설을 공유할 수 있어, 서울·경기·인천 전체를 하나의 숫자로 단순 합산하면 중복이 생긴다. 따라서 아래처럼 `운영사가 공개한 네트워크 규모`와 `개별 시설까지 확인한 목록`을 분리한다.

핵심 수치만 보면 트니트니 출강 문화센터는 수도권 365곳이다. 다만 경기와 인천은 운영사가 합산 수치만 공개한다. 공공 놀이·체험 네트워크는 서울 117곳, 경기 77곳·92곳의 서로 다른 사업, 인천 60곳이 확인되지만 네트워크 간 중복을 제거하지 않았으므로 서로 더하지 않는다.

## 신뢰도와 사용 시 주의사항

1. 트니트니 365곳은 운영사의 공식 통계라 규모 파악 신뢰도는 높지만, 경기와 인천의 개별 지점 목록은 공개하지 않는다.
2. 서울·경기의 문화센터 개별 목록은 2026 여름학기 공식 강좌 검색에서 직접 확인한 최소 목록이다. 다른 유통사·주민센터까지 포함한 전수 목록은 아니다.
3. 경기 아이사랑놀이터 92곳은 2025-07-10 기준 최신 확인 공식 수치이므로 2026년 실제 운영 수와 다를 수 있다.
4. 공공 놀이시설·박물관·직영센터가 같은 복합건물에 있을 수 있어 네트워크 수치를 서로 합산하지 않는다.
5. 가격·예약·휴관 정보는 바뀔 수 있으므로 실제 방문 전 각 공식 페이지에서 다시 확인해야 한다.

## 간단한 결론

수도권 트니트니 출강 문화센터는 공식 통계상 365곳이다. 여기에 서울·경기·인천의 공공 놀이·체험 네트워크와 대표 체험관이 별도로 존재하지만, 중복 가능성 때문에 하나의 총합으로 합치지 않는 것이 정확하다. 실제 콘텐츠 시드나 장소 DB로 옮길 때는 이 문서의 개별 확인 목록을 시작점으로 삼고, 네트워크별 전체 지점은 주소 기준으로 추가 수집·중복 제거해야 한다.
