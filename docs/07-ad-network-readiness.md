# 07차: 광고 네트워크(AdSense/네이버/카카오) 준비

> 상세화 전 검토 체크포인트  
> - 네트워크별 심사 요건과 사이트 현재 상태의 격차가 파악되었는가?  
> - 계정/정책/검수 항목이 팀 내 책임 단위로 분리되었는가?  
> - 실제 삽입 전 필요한 증빙(콘텐츠 수, 정책 페이지 등) 기준이 있는가?

## 목적

- 광고 네트워크 심사 통과를 위한 준비 항목을 구조화한다.
- 계정 생성, 정책 준수, 검수 준비의 작업 경계를 정의한다.
- 08차 삽입 단계에서 재작업을 줄이기 위한 사전 기준을 고정한다.

## 범위(포함/제외)

- 포함:
  - 네트워크별 준비 체크리스트 골격
  - 계정/정책/검수 준비 항목 정리
  - 광고 슬롯 인벤토리 검증 항목 정리
- 제외:
  - 실계정 심사 제출 실행
  - 광고 코드 배포

## 선행조건

- 03차 광고 슬롯 정의가 필요하다.
- 06차 SEO/기본 품질 작업 방향이 정리되어야 한다.

## 핵심 작업

- 네트워크별 필수 준비 항목 정리
- 정책 페이지(개인정보/문의 등) 준비 여부 점검 항목 정의
- 슬롯별 정책 위험도 사전 점검 틀 작성
- 심사 제출 전 최소 품질 기준 합의

## 산출물

- 네트워크 준비 체크리스트
- 계정/정책/검수 준비 상태표
- 슬롯 인벤토리 정책 검증표

## 의사결정 포인트

- 네트워크 동시 진행 vs 순차 진행
- 초기 적용 네트워크 우선순위
- 심사 실패 시 보완 반복 주기

## 리스크

- 외부 광고 정책 변경으로 문서 최신성 저하 가능
- 콘텐츠/정책 페이지 미비 시 심사 지연 가능

## 완료 기준

- 네트워크별 준비 항목이 문서화됨
- 심사 제출 전 체크해야 할 항목이 명확함
- 08차 삽입 단계로 넘길 슬롯 검증 정보가 준비됨

## 다음 단계 인계

- 08차에서 실제 삽입에 사용할 네트워크 우선순위 전달
- 정책 검토가 필요한 슬롯 목록 전달
