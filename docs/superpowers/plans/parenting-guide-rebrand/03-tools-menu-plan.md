# Tools Menu Plan

## 목표

도구 메뉴를 장소 탐색 뒤에 자연스럽게 이어지는 두 번째 메인 허브로 정리한다.

## 역할

부모가 읽은 정보를 실제 비용/연령/조건 판단으로 전환하는 허브다.

## 1차 구성

- 연령/성장 계산
- 비용/예산 계산
- 세금/증여 계산

## 1차 공개 후보

1. 연령/개월수 계산기
2. 가족 나들이 예산 계산기
3. 키즈카페·체험 월예산 계산기
4. 지역 혜택 자가진단 도구
5. 성장 계산기
6. 자녀 비과세 증여 계산기
7. 어린이집·유치원 비용 계산기

## 허브 요구사항

- 카테고리별 묶음
- 대표 도구 카드
- 관련 장소 글 연결
- 관련 혜택 글 연결

## 도구 공통 규칙

- 도구 결과에서 다음 행동 링크 제공
- 각 도구는 글 세트와 같이 운영
- 의료성 오해 가능 도구는 제외

## 구현 전 결정할 것

- 1차 공개 도구 수
- 기존 금융 도구와 육아 도구의 공존 방식
- 도구 허브 카피와 카드 스타일

간단한 결론:

도구 메뉴는 독립성이 있어야 하지만, 장소 탐색과 끊어지면 안 된다.
