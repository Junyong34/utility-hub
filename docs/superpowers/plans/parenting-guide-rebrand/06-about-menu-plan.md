# About Menu Plan

## 목표

소개 메뉴를 브랜드 선언과 신뢰 장치 역할로 정리한다.

## 역할

왜 이 사이트가 존재하는지, 무엇을 도와주는지, 어떤 방식으로 정보를 정리하는지를 설명한다.

## 핵심 메시지

- 이 사이트는 부모의 시간을 아끼기 위해 존재한다
- 지역, 연령, 비용, 혜택을 같이 본다
- 과장된 후기보다 실전 판단을 돕는다

## 포함할 요소

- 서비스 한 줄 설명
- 어떤 정보를 다루는지
- 어떤 정보를 다루지 않는지
- 업데이트 원칙
- 면책/주의 문구
- `by Zento` 표기 정책

## 구현 전 결정할 것

- 최종 브랜드명
- 브랜드 스토리 길이
- 신뢰 요소 노출 범위

간단한 결론:

소개 페이지는 감성 브랜딩보다 `무엇을 잘하고 무엇은 안 하는지`를 분명히 보여줘야 한다.
