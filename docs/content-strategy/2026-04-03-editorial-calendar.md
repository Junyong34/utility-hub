# Zento 3개월 편집 캘린더

작성일: 2026-04-03  
목표: `비교·절약·실전 의사결정` 포지셔닝에 맞춰 검색 유입과 도구 전환을 동시에 노린다.

## 운영 원칙

1. 검색량보다 `의사결정 의도`가 강한 주제를 우선한다.
2. 글만 읽고 끝나지 않게, 가능한 경우 관련 도구로 연결한다.
3. 공식 기관·공개 자료가 있는 주제를 우선해 신뢰도를 높인다.
4. 같은 주제를 `가이드 글 + 계산기/비교 도구` 쌍으로 운영한다.
5. 공항·역 주차, 금융 계산, 소비자 비교처럼 반복 검색이 일어나는 축을 키운다.

## 우선순위 기준

- `P0`: 바로 써먹는 고의도 주제, 도구 연결 가능, 빠른 발행 가치 높음
- `P1`: 유입 가능성 높고 내부 링크 확장에 유리
- `P2`: 브랜드 보강용, 계절성·보완성 주제

## 상위 5개 선정 이유

1. 검색 의도가 명확하다.
2. 현재 사이트의 강점인 `주차 + 계산기` 축을 바로 살릴 수 있다.
3. 기존 문구 리포지셔닝과 가장 잘 맞는다.
4. 글 안에서 도구 클릭 전환을 만들기 쉽다.
5. 내부 링크 구조를 빠르게 강화할 수 있다.
