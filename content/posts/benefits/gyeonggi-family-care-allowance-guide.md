## 지원 대상과 신청 기간 요약

이 글은 **2026년 5월 26일 기준** 경기도 가족돌봄수당 안내입니다. 경기도 공식 안내에 따르면 2026년 가족돌봄수당은 참여 시군이 14개에서 26개로 확대되었습니다. 생후 24~36개월 영유아를 친인척이나 이웃이 돌보면 아동 수에 따라 월 30만~60만 원을 지원합니다.

| 먼저 볼 항목     | 2026년 기준 확인 내용                                |
| ---------------- | ---------------------------------------------------- |
| 아이 나이        | 공식 안내상 생후 24~36개월 구간을 먼저 확인합니다.   |
| 주소지 참여 여부 | 참여 시군과 신청 시작 시점이 다를 수 있습니다.       |
| 돌봄 조력자      | 4촌 이내 친인척 또는 이웃 돌봄 여부를 확인합니다.    |
| 신청 기간        | 경기도 안내 기준 매월 1~15일 신청 기간을 확인합니다. |

## 먼저 확인할 가정

어린이집 등원 전후나 방학, 부모 근무 시간 때문에 돌봄 공백이 생긴다면 경기도 가족돌봄수당을 먼저 확인하세요.

아래 상황이면 먼저 확인하세요.

| 상황                                | 확인할 것                                               |
| ----------------------------------- | ------------------------------------------------------- |
| 조부모가 정기적으로 아이를 돌봄     | 친인척 돌봄 인정 범위와 활동 시간                       |
| 맞벌이로 돌봄 공백이 있음           | 신청 자격과 증빙 기준                                   |
| 경기 남부·북부 안에서 이사 예정     | 전입·거주지 기준과 참여 시군                            |
| 24개월 이후 아이 돌봄 비용이 늘어남 | 어린이집, 아이돌봄서비스, 가족돌봄수당의 중복·전환 기준 |

## 신청 전에 확인할 조건

가족돌봄수당은 “경기도에 살면 누구나 자동 지급”되는 혜택이 아닙니다. 공식 안내와 신청 화면에서 아래 항목을 확인해야 합니다.

| 체크 항목    | 설명                                                                             |
| ------------ | -------------------------------------------------------------------------------- |
| 참여 시군    | 2026년에는 26개 시군으로 확대되었지만, 시군별 신청 시작 시점이 다를 수 있습니다. |
| 아이 개월 수 | 생후 24~36개월 구간인지 먼저 봅니다.                                             |
| 돌봄 조력자  | 4촌 이내 친인척 또는 이웃 주민 돌봄인지 확인합니다.                              |
| 신청 기간    | 매월 신청 가능 기간이 정해져 있으므로 월 초에 확인하세요.                        |
| 지급 방식    | 돌봄 활동 확인 후 다음 달 지급되는지 신청 화면에서 확인합니다.                   |

## 어린이집·아이돌봄서비스와의 차이

이 혜택은 어린이집 보육료처럼 기관에 바로 결제되지 않습니다. 가정의 돌봄 공백을 줄이기 위한 지역 지원이므로 아래 항목을 함께 비교하세요.

| 비교 항목          | 볼 점                                                      |
| ------------------ | ---------------------------------------------------------- |
| 어린이집 보육료    | 낮 시간 기관 이용 비용을 줄이는 지원인지 확인합니다.       |
| 아이돌봄서비스     | 정부지원 자격과 본인부담금을 따로 봅니다.                  |
| 가족돌봄수당       | 가족·이웃 돌봄이 정기적으로 발생하는지 봅니다.             |
| 지역 공공 놀이공간 | 돌봄 조력자와 함께 갈 수 있는 가까운 실내 공간을 찾습니다. |
