## 이용 조건과 할인 혜택 요약

이 글은 **2026년 5월 26일 기준** 서울형 키즈카페와 서울형키즈카페머니 이용 안내입니다. 서울에서 아이와 갈 실내 공간을 찾는다면 공공 놀이공간인 서울형 키즈카페를 먼저 확인하세요. 민간 키즈카페를 이용할 때는 서울형키즈카페머니 사용처를 함께 확인하면 비용을 줄일 수 있습니다.

| 먼저 볼 항목       | 확인할 내용                                                         |
| ------------------ | ------------------------------------------------------------------- |
| 공공 키즈카페 이용 | 서울형 키즈카페 예약 가능 지점과 이용 연령을 확인합니다.            |
| 민간 키즈카페 이용 | 서울형 인증 민간 키즈카페와 서울형키즈카페머니 사용처를 확인합니다. |
| 주말·우천 시 방문  | 예약 마감이 빠를 수 있으므로 날짜와 회차를 먼저 확인합니다.         |
| 비용 할인          | 서울페이플러스 앱에서 서울형키즈카페머니 발행 여부를 확인합니다.    |

## 먼저 확인할 가정

서울형 키즈카페는 날씨와 계절 영향을 덜 받는 실내 놀이공간을 찾는 가정에 특히 잘 맞습니다. 서울시 공식 안내에서는 서울형 키즈카페 사업을 0~9세 아동과 보호자를 위한 놀이공간으로 설명합니다.

아래 상황이라면 먼저 확인하세요.

| 상황                                    | 확인 내용                                       |
| --------------------------------------- | ----------------------------------------------- |
| 0~9세 아이와 서울 실내 공간을 찾는 경우 | 집 근처 서울형 키즈카페 지점과 이용 가능 연령   |
| 비 오는 날, 미세먼지 심한 날            | 실내 회차 예약 가능 여부                        |
| 민간 키즈카페 비용이 부담되는 경우      | 서울형키즈카페머니 사용처와 할인 구매 가능 여부 |
| 아이와 짧게 다녀올 곳을 찾는 경우       | 회차 시간, 보호자 동반 기준, 현장접수 가능 여부 |

## 예약 전에 확인할 것

서울형 키즈카페는 지점별 운영 방식이 다를 수 있습니다. 예약하기 전에 아래 항목을 확인하세요.

| 체크 항목   | 확인할 이유                                                   |
| ----------- | ------------------------------------------------------------- |
| 이용 연령   | 지점마다 영유아 중심인지, 4~9세 중심인지 달라질 수 있습니다.  |
| 예약 방식   | 사전 예약 중심인지, 일부 현장접수가 가능한지 확인해야 합니다. |
| 회차 시간   | 아이 낮잠, 식사 시간과 겹치면 실제 만족도가 떨어집니다.       |
| 보호자 기준 | 보호자 입장 인원과 동반 기준을 지점별로 봐야 합니다.          |
| 주차·유모차 | 공공시설 안에 있더라도 주차가 항상 편한 것은 아닙니다.        |

## 서울형키즈카페머니 사용처와 구매 방법

서울형키즈카페머니는 서울시가 선정한 서울형 인증 민간 키즈카페 전용 서울페이 상품권입니다. 공식 안내에 따르면 서울페이플러스 앱에서 서울형키즈카페머니 상품권을 20% 할인된 가격으로 구매할 수 있습니다.

다만 이 혜택은 “모든 키즈카페에서 자동 할인”이 아닙니다. 사용 전에 아래 순서로 확인하세요.

1. 서울형키즈카페머니 사용처 목록에서 방문하려는 키즈카페를 찾습니다.
2. 서울페이플러스 앱에서 해당 상품권 발행·잔여 여부를 확인합니다.
3. 키즈카페 예약 또는 방문 전에 상품권 결제 가능 여부를 다시 확인합니다.
4. 아이 연령 제한, 보호자 요금, 추가 이용료가 있는지 봅니다.
