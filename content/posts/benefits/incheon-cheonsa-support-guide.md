## 한눈에 요약

이 글은 **2026년 5월 26일 기준**으로 인천 천사지원금을 정리한 글입니다. 인천시 공식 안내에 따르면 천사지원금은 인천형 출생정책 `아이플러스(i+) 1억드림`의 일부로, 2023년생 아이부터 1~7세까지 연 120만 원씩 인천e음 포인트로 지원하는 제도입니다.

| 먼저 볼 질문                 | 2026년 기준 확인 포인트                                       |
| ---------------------------- | ------------------------------------------------------------- |
| 아이 출생연도가 해당하나요?  | 2023년 1월 1일 이후 출생아부터 확인합니다.                    |
| 아이 나이가 맞나요?          | 1세~7세 구간을 봅니다.                                        |
| 인천 거주 요건을 충족하나요? | 아동 생일 기준 인천시 계속 거주 요건을 확인합니다.            |
| 언제 신청하나요?             | 공식 안내상 아동 생일 이후 120일 이내 신청 기준을 확인합니다. |

## 누가 먼저 보면 좋나

인천 천사지원금은 인천에 거주하는 1~7세 아이 가정이 먼저 확인해야 할 지역 혜택입니다. 출생 직후 받는 첫만남이용권과 달리, 아이가 자라는 동안 매년 확인하는 구조라 생일 기준 신청 시점을 놓치지 않는 것이 중요합니다.

아래 상황이면 먼저 확인하세요.

| 상황                           | 확인할 것                              |
| ------------------------------ | -------------------------------------- |
| 2023년 이후 출생한 아이가 있음 | 연령과 출생연도 기준                   |
| 인천 안에서 계속 거주 중       | 부 또는 모와 아동의 주민등록 요건      |
| 인천으로 전입했음              | 전입 아동의 거주 요건과 신청 가능 시점 |
| 첫 신청을 앞두고 있음          | 정부24 신청 경로와 인천e음 가입 여부   |

## 신청 전에 확인할 조건

천사지원금은 지역화폐 인천e음 포인트로 지급되는 혜택입니다. 신청 전에 아래 항목을 확인하세요.

| 체크 항목 | 설명                                                                                         |
| --------- | -------------------------------------------------------------------------------------------- |
| 지급 대상 | 2023년 1월 1일 이후 출생아, 1~7세 구간인지 확인합니다.                                       |
| 거주 요건 | 아동의 생일 기준 부 또는 모와 주민등록을 같이 두고 인천시에 1년 이상 계속 거주했는지 봅니다. |
| 신청 기한 | 아동 생일 이후 120일 이내 신청 기준을 확인합니다.                                            |
| 신청 경로 | 원칙적으로 정부24 온라인 신청을 사용합니다.                                                  |
| 방문 신청 | 부 또는 모 외 보호자 신청 등 온라인 신청이 어려운 경우 행정복지센터 방문 가능 여부를 봅니다. |

## 첫만남이용권·부모급여와 같이 볼 점

천사지원금은 국가 공통 지원과 별도로 보는 인천 지역 혜택입니다. 아이의 나이에 따라 아래처럼 나눠 보면 이해하기 쉽습니다.

| 아이 기준            | 같이 볼 혜택                         |
| -------------------- | ------------------------------------ |
| 출생 직후            | 첫만남이용권, 부모급여, 아동수당     |
| 0~23개월             | 부모급여, 아동수당, 보육료 전환 여부 |
| 1~7세 인천 거주      | 천사지원금, 인천e음 포인트 사용처    |
| 어린이집·유치원 이용 | 보육료, 유아학비, 아동수당           |

천사지원금은 연령과 주소지 기준이 핵심입니다. 부모급여나 아동수당처럼 전국 공통 제도와 신청 기준이 다르므로, 한 번에 신청된다고 가정하지 말고 별도로 확인하는 편이 안전합니다.

## 마무리

인천 천사지원금은 아이 출생연도, 생일 기준 신청 시점, 인천 거주 요건을 함께 봐야 하는 지역 혜택입니다. 2026년에는 신청 기간과 반복 신청 방식이 개선되었지만, 실제 신청 가능 여부는 정부24와 인천시 공식 안내 화면에서 다시 확인하는 것이 안전합니다.
